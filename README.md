# Code Wiz

This repository holds **all the resources for Code Wiz**—curriculum materials, practice exercises, and code samples used for teaching and learning.

## Contents

- **Basic Java** — Variables, strings, conditionals, loops, arrays, and more
- **Advanced Java** — Methods, parameters, accumulators, and beyond
- **Basic Python** — Strings, loops, and foundational Python concepts

Use these files as reference, practice, or teaching material for Code Wiz programs.

## Contributing

Add new lessons, exercises, or fixes by opening a pull request or pushing to the appropriate folder.
